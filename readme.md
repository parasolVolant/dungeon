But :
Atteindre la dernière pièce du donjon (en bas à droite).

Génération de donjon : 
Le labyrinthe est généré à l'aide d'un algorithme d'arbre couvrant.

Condition de défaite :
Le joueur meurt.

Commandes :
Les flèches : se déplacer.
I : ouvrir/fermer l'inventaire.
U : ouvrir/fermer la fenêtre de statut (affiche vie et force du joueur).
A : équiper un objet.
Z : jeter un objet.

Nouvelle partie :
Pour démarrer une nouvelle partie il faut redémarrer le projet.
